def nurbswb.scanbackbonecut.run (   model = 'shoeAdam',
  point_cloud = 'shoe_last_scanned',
  showpoints = True,
  showxywire = True,
  showxypoints = True 
)

For each rib of the backbone a point set is calculated.

  • the starting point cloud is a scanned shoe last asc file
  • point sets and the corresponding rib sketches are grouped into folders GRP_1, ...
  • the folder ScanpointsSource contains all point slices on their real position
  • the folder clones2 contains clones of the sketcher ribs on their right position
Parameters
model- name of the shoe model
point_cloud- name of the pount cloud document
showpoints- display the point sets
showxywire- create a wire which approximates the point set as mapping in the xy-plane
showxypoints- display the mapping of the point sets into the xy-plane

. 

create slices of the pointcloud near the ribs
